# Training Timer App

Aplicación web de temporizador de entrenamiento construida con **Vite + React + Tailwind CSS**.

## Scripts

- `npm install` instala dependencias.
- `npm run dev` levanta el entorno local.
- `npm run build` genera build de producción.
- `npm run preview` previsualiza la build.

## Stack

- React 18
- Vite 5
- Tailwind CSS 3
- framer-motion
- lucide-react
